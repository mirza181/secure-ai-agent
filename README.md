# Secure AI Academic Assistant

An AI-powered chatbot built using Python and Streamlit to help CS/BCA students learn important computer science topics.

## Features
- Chatbot for Computer Science questions
- Knowledge base for BCA subjects
- Smart keyword search
- Suggestion buttons for common questions
- Modern Streamlit UI

## Technologies Used
- Python
- Streamlit
- HTML/CSS
- Git & GitHub

## How to Run

1. Install requirements
2. Run the app

```bash
streamlit run app.py
